## Cybersecurity Journey

My journey into cybersecurity started with a simple question:

**"How do systems actually break?"**

That curiosity led me into hands-on labs, security challenges, writeups, and automation projects.

Instead of focusing only on theory, I prefer learning through experimentation, documenting findings, and understanding why vulnerabilities exist.

Over time, I developed a strong interest in offensive security, web application testing, privilege escalation, and workflow automation.

My current goal is to become a professional Penetration Tester while continuing to build tools, publish writeups, and contribute to the security community.
